<script src="bootstrap/js/bootstrap.bundle.min.js"></script>

2. Bootstrap Structure and Files

2.1: Structure explanation
  • css/ folder: Bootstrap's stylesheets (e.g., bootstrap.min.css)
  • js/ folder: Bootstrap JavaScript files (e.g., bootstrap.bundle.min.js includes Popper.js)
  • icons/: Bootstrap Icons SVG files (optional)
2.2: Including Bootstrap JS Bundle

Included below before </body>:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
